**Lab 03 JS Canvas – Systemy Cząsteczkowe**

**Opis**
Zastosowałem podejście obiektowe do stworzenia dynamicznej symulacji cząsteczkowej przy użyciu Canvas API. Każdy fajerwerk jest oddzielnym obiektem, który po osiągnięciu celu generuje chmurę cząsteczek o losowych kolorach (HSL). Zaimplementowałem fizykę ruchu uwzględniającą grawitację, opór powietrza oraz odbijanie się cząsteczek od dolnej krawędzi ekranu. Projekt zawiera interaktywne suwaki pozwalające na dynamiczną zmianę siły grawitacji oraz liczby generowanych cząsteczek w czasie rzeczywistym. Pociski sa wystrzeliwane same ale można też wybrac miejscie dla wystrzalu własnej fajerwerki.
